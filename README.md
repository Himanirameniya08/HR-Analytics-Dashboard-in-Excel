# HR Analytics Dashboard in Excel
## Project Objective

To build an interactive HR Analytics Dashboard in Excel that transforms raw employee data into meaningful insights for better workforce decision-making. The goal was to analyze key HR metrics like headcount, salary, hiring trends, and diversity.
# Key Questions (KPIs)
- What is the total workforce size?
- What is the gender distribution (%)?
- How are employees distributed across departments?
- What is the average salary overall and by department?
- What are the hiring trends over time?
- How are employees distributed based on performance ratings?

# Process
1. Data Cleaning & Preparation
- Removed inconsistencies and structured raw data
- Ensured correct data types (date, salary, categories)
2. Data Analysis
- Created Pivot Tables for KPI calculations
- Aggregated data by department, gender, and time
3. Dashboard Creation
- Built interactive charts (bar, line, donut, etc.)
- Added slicers for dynamic filtering (Year, Month, etc.)
Designed a clean and consistent dashboard theme
4. Visualization & Storytelling
- Converted raw numbers into easy-to-understand visuals
- Highlighted key insights for decision-making
